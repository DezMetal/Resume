# AirControl

Gesture-based computer control via Leap Motion sensor.

## Overview
Control your computer with hand gestures. The Leap Motion sensor tracks hand poses in real-time, and a web UI lets you train custom gestures to trigger any action.

## Key Features
- **Leap Motion integration** - hardware SDK for hand tracking
- **Custom pose training** - teach the system your gestures
- **Action mapping** - connect gestures to system commands
- **Web configuration UI** - browser-based setup
- **Real-time recognition** - responsive gesture detection

## Technologies
- Python | Leap Motion SDK | Gesture recognition | Flask web UI
