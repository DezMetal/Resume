# DSS - D-Net Signature Stylesheet

Custom geometric CSS framework with theming support.

## Overview
A signature design system used across D-Net Lab projects. Features geometric cut corners, glow effects, and full light/dark theming. This resume is styled with DSS.

## Key Features
- **Geometric aesthetics** - cut corners, angular shapes
- **Glow effects** - accent color glow on interactive elements
- **Light/dark theming** - CSS variable-based theme switching
- **Modular components** - cards, buttons, forms, navigation
- **Background patterns** - grid and dot textures

## Technologies
- CSS | CSS variables | Geometric design | Theme system
