# D-Net Platform

Multi-tenant web application with AI personas, content management, and Stripe integration.

## Overview
A complete web platform supporting AI-powered interactions, tiered user subscriptions, and seller storefronts. Demonstrates full-stack development with payment processing, multi-tenancy, and LLM integration.

## Key Features
- **Multi-tenant architecture** - isolated user spaces with role-based access
- **AI personas** - configurable LLM-powered interactions
- **Stripe integration** - subscription tiers for users, storefronts for sellers
- **Content management** - flexible content creation and publishing
- **Gemini API** - intelligent content generation and chat

## Technologies
- Python | Flask | SQLAlchemy | Stripe API | Gemini | Jinja2
