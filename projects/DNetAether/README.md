# DNetAether

Voice productivity tool with hotkey-triggered transcription and clipboard output.

## Overview
Press a hotkey, speak, release - your words appear in the clipboard ready to paste. Multiple modes allow raw dictation, LLM text refinement, natural-language-to-code generation, and conversational AI with voice feedback.

## Key Features
- **Offline transcription** via Whisper (no cloud dependency for speech-to-text)
- **Multiple modes**: Dictation, Flow (LLM refinement), Code generation, Chat
- **Clipboard-first output** - seamless integration with any application
- **Volume ducking** - automatically lowers other audio during speech operations
- **StreamAudio integration** via git submodule for TTS feedback

## Technologies
- Python | Whisper | Gemini API | Piper TTS | pynput (global hotkeys)
