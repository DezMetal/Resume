# IsekaiAdventure

AI language learning RPG - learn Japanese through immersive story.

## Overview
An experimental approach to language learning: engage with an AI-driven narrative that teaches Japanese through context and immersion. Intelligent context management maintains story coherence across long conversations.

## Key Features
- **Immersive narrative** - learn through story, not drills
- **Context management** - coherent long-form storytelling
- **Language integration** - Japanese woven into gameplay
- **AI-driven** - dynamic story responds to player
- **Vocabulary building** - natural acquisition through use

## Technologies
- Python | Gemini API | Context window management | Interactive fiction
