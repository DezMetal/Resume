# VirtualLense

Screen region analysis with configurable overlays.

## Overview
Capture and analyze specific screen regions in real-time. Extract text via OCR, detect changes for monitoring, and overlay results directly on screen. Useful for automation triggers and accessibility.

## Key Features
- **Region selection** - define any screen area to monitor
- **OCR text extraction** - pull text from screen content
- **Change detection** - trigger on visual differences
- **Overlay windows** - display results on top of screen
- **Automation-ready** - connect to other systems

## Technologies
- Python | EasyOCR | OpenCV | Tkinter | Screen capture
