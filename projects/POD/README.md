# POD Pipeline

Print-on-demand automation - images in, store full of products out.

## Overview
Drop images into a folder and watch your store fill with products. This pipeline orchestrates AI image processing (ComfyUI) with product deployment (Printify) to automate the entire print-on-demand workflow. Currently serving paying clients.

## Key Features
- **Folder watch automation** - drop images, get products
- **ComfyUI integration** - AI-powered image processing
- **Printify API** - automatic product creation and publishing
- **Batch processing** - handle multiple images efficiently
- **WebSocket status updates** - real-time progress feedback

## Technologies
- Python | ComfyUI API | Printify API | Watchdog | WebSocket
